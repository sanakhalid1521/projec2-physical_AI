# جسمانی ای آئی کا تعارف

جسمانی ای آئی ایک ایسا میدان ہے جو مصنوعی ذہانت کو جسمانی نظاموں کے ساتھ جوڑتا ہے۔ یہ جسمانی دنیا کے ساتھ سینسرز اور ایکٹو ایٹرز کے ذریعے تعامل کرنے والے ذہیب ایجنٹس کو تیار کرنے پر مرکوز ہے۔

## جسمانی ای آئی کیا ہے؟

جسمانی ای آئی روایتی ای آئی سے اس لحاظ سے مختلف ہے کہ یہ دنیا کی جسمانی خصوصیات کو مدنظر رکھنا چاہیے۔ اس میں زور دیا جاتا ہے کہ طاقت، حرکت، مٹھائی اور گراؤنٹی جیسے تصورات کو سمجھنا ہوگا۔

## اہم تصورات

- جسمانیت: جسمانی شکل والے ای آئی نظام
- حسی محرک سیکھنا: ماحول کے ساتھ تعامل کے ذریعے سیکھنا
- حقیقی دنیا کے محدود کن: جسمانی قوانین جو رویے کو متاثر کرتے ہیں

## عملی سرگرمی

فزکس انجن کا استعمال کرتے ہوئے ایک سادہ شبیہہ تیار کریں جس میں ایک گیند اچھلتی ہے۔ دیکھیں کہ گراؤنٹی اور لچک جیسے پیرامیٹرز کو تبدیل کرنے سے رویہ کیسے متاثر ہوتا ہے۔

```javascript
// مثال کے طور پر فزکس کی تبدیلی
const ball = {
  position: { x: 0, y: 10 },
  velocity: { x: 0, y: 0 },
  gravity: 9.8,
  bounceFactor: 0.8
};
```

گیند کی حرکت کے لیے فزکس کے مساوات کو لاگو کریں اور دیکھیں کہ مختلف پیرامیٹرز کے ساتھ یہ کیسے مختلف طرز عمل کرتا ہے۔
